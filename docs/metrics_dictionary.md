# Metrics Dictionary

| Metrica | Definicao | Uso executivo |
|---|---|---|
| ARR novo | Soma de `new_arr` em revenue | Mede receita recorrente criada pela aquisicao. |
| Spend | Soma de `spend_amount` | Base de investimento de marketing por canal/campanha. |
| CAC | Spend dividido por novos clientes | Avalia custo de aquisicao por cliente. |
| CAC Payback | Spend dividido por margem bruta mensal do ARR | Mostra em quantos meses o investimento volta. |
| LTV/CAC | ARR vezes margem e retencao dividido por spend | Aproxima retorno unitario da aquisicao. |
| Lead-to-customer | Clientes divididos por leads | Mede qualidade do funil por canal. |
| Retencao 90d | Media de `retained_90d` | Evita escalar canais que compram churn. |
| NRR 90d | Media de `nrr_90d` | Mostra expansao/contracao inicial por coorte. |
| First-touch attribution | Receita creditada ao primeiro contato | Boa para avaliar captura inicial de demanda. |
| Last-touch attribution | Receita creditada ao ultimo contato | Boa para avaliar conversao final. |
| Multi-touch equal | Receita dividida entre contatos | Base operacional mais balanceada para budget. |
